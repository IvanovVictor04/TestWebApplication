Это мое веб приложение, которое имитирует сайт с различными моделями машин, самолетов и т.д.
В нем я реализовал основной функционал, который присутствует на большинстве реальных сайтах а также возможность добавить новую модель.
Источник для моедели MyProducts я взял из созданной базы данных classic models в MySql. Сама база данных большая, 
поэтому я прикрепляю ссылку(https://raw.githubusercontent.com/harsha547/ClassicModels-Database-Queries/master/database.sql) откуда можно скопировать только нужную таблицу, после чего нужно лишь изменить connection string.
